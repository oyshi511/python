üåÄ What is a Loop?

A loop lets your program repeat actions automatically ‚Äî instead of writing the same line again and again.

üîπ Types of Loops in Python

There are mainly two types:

  • for loop ‚Äì repeats a specific number of times.
  • while loop ‚Äì repeats as long as a condition is true.

For loop

The range() function helps us decide how many times to repeat.

Example: for i in range(5): print("I love Python")

range(5) means it will count from 0 to 4 (that‚Äôs 5 times total). i is just a variable that changes every loop: 0, 1, 2, 3, 4.

While loop

A while loop keeps running as long as a condition is true.

Example: count = 1 while count <= 5: print("Hello, Python!") count = count + 1

How it works?

  1. It starts with count = 1
  2. Checks the condition count <= 5
  3. Prints the line
  4. Adds 1 to count
  5. Repeats until count becomes 6 ‚Üí condition is false, loop stops
If you forget to update the variable (like count = count + 1), the loop will never stop ‚Äî it becomes an infinite loop.

List

A list is a way to store multiple values in a single variable. It‚Äôs like a box that holds many items together.

Example: fruits = ["apple", "banana", "cherry"] print(fruits)
Output: ['apple', 'banana', 'cherry']

Each item inside the brackets [ ] is separated by a comma.

üîπ Accessing items from a list

You can pick items by index number ‚Äî counting starts at 0!

print(fruits[0]) # apple print(fruits[1]) # banana print(fruits[2]) # cherry

üîπ Changing or adding items

fruits[1] = "mango" # change ‚Äúbanana‚Äù to ‚Äúmango‚Äù fruits.append("orange") # add ‚Äúorange‚Äù to the end print(fruits)
Output: ['apple', 'mango', 'cherry', 'orange']

Using loops with lists

You can loop through all items automatically!

1. Adding items to a list

You can add new elements using .append()

friends = ["Ava", "Liam", "Noah"] friends.append("Emma") print(friends)
output: ['Ava', 'Liam', 'Noah', 'Emma']

2. Removing items

Use .remove() or .pop()

friends.remove("Liam") # removes by name print(friends) friends.pop() # removes the last one print(friends)

3. Counting and sorting

You can check the list length or alphabetically order items:

print(len(friends)) # counts items friends.sort() # sorts alphabetically print(friends)

4. Making a list from user input

You can ask the user to type several items and build a list as they go!

items = [] for i in range(3): item = input("Enter an item: ") items.append(item) print("Your list:", items)

Nested Loops and Combined Logic

What is Nested Loop

A nested loop means one loop inside another. You used it when you need to repeat something within a repetition for example, printing patterns, comparing lists, or checking conditions.

Example: for i in range(3): for j in range(2): print("i =", i, "j =", j)

The outer loop controls rows and the inner loop controls columns or repeated actions within each row
