Introduction to Python Programming

💡 What Is Python?

Imagine you want to talk to your computer and tell it things like:

“Hey, add these numbers” or “Show me my favorite color.”

But computers don’t understand Bangla or English — they understand code. Python is a programming language that helps you talk to your computer in simple, English-like commands.

🧠 Why Do People Learn Python?

Because Python is:

  • 🧩 Easy to learn — perfect for beginners
  • 🌍 Used everywhere — websites, apps, AI, data, and games
  • ⚡ Powerful — used by Google, NASA, Netflix, and more
  • 💼 Good career choice — Python developers earn a lot

⚙️ How Python Works

  • print() — tells Python to show something on the screen
  • Whatever you put inside quotes ("") is text (called a string)
  • Python then shows that text as output

🧠 What Are Variables?

A variable is like a small box with a name that stores information.

  • name = "Oyshi" → text (string)
  • age = 18 → number (integer)

🧩 Rule: How to Create a Variable

variable_name = value

city = "Barisal"
study_hours = 14
goal = "MIT Scholarship"

You can use variables inside print(). Python automatically adds spaces when you separate things with commas.

🔤 Types of Data in Python
Type Example Meaning
String (str) "Hello" Text (inside quotes)
Integer (int) 18 Whole number
Float (float) 9.5 Decimal number

type() tells you what kind of data is stored in a variable.

🗣️ The input() Function

input() allows your program to interact with people like a mini chatbot 🤖

name = input("What is your name? ")
print("Hello,", name)

Note: input() always returns text (string), even if you type a number.

Example:

age = input("How old are you? ")
print("You are", age, "years old.")

Here, age is still a string. To do math with it, you must convert it to a number.

🔢 Converting Input to Numbers

To convert text input into numbers:

  • int() → changes to integer (whole number)
  • float() → changes to decimal number

🧠 Making Decisions — if, elif, else

Now your program can think using conditions.

Syntax:

if condition:
    # do this
else:
    # do this instead

Example:

age = int(input("Enter your age: "))

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

⚙️ Multiple Conditions with elif

marks = int(input("Enter your marks: "))

if marks >= 80:
    print("Grade: A+")
elif marks >= 70:
    print("Grade: A")
elif marks >= 60:
    print("Grade: B")
else:
    print("Grade: Fail")

💡 elif means “else if.”
