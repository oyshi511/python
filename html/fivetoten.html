🌀 What is a Loop?

A loop lets your program repeat actions automatically — instead of writing the same line again and again.

🔹 Types of Loops in Python

There are mainly two types:

  • for loop – repeats a specific number of times.
  • while loop – repeats as long as a condition is true.

For loop

The range() function helps us decide how many times to repeat.

Example: for i in range(5): print("I love Python")

range(5) means it will count from 0 to 4 (that’s 5 times total). i is just a variable that changes every loop: 0, 1, 2, 3, 4.

While loop

A while loop keeps running as long as a condition is true.

Example: count = 1 while count <= 5: print("Hello, Python!") count = count + 1

How it works?

  1. It starts with count = 1
  2. Checks the condition count <= 5
  3. Prints the line
  4. Adds 1 to count
  5. Repeats until count becomes 6 → condition is false, loop stops
If you forget to update the variable (like count = count + 1), the loop will never stop — it becomes an infinite loop.

List

A list is a way to store multiple values in a single variable. It’s like a box that holds many items together.

Example: fruits = ["apple", "banana", "cherry"] print(fruits)
Output: ['apple', 'banana', 'cherry']

Each item inside the brackets [ ] is separated by a comma.

🔹 Accessing items from a list

You can pick items by index number — counting starts at 0!

print(fruits[0]) # apple print(fruits[1]) # banana print(fruits[2]) # cherry

🔹 Changing or adding items

fruits[1] = "mango" # change “banana” to “mango” fruits.append("orange") # add “orange” to the end print(fruits)
Output: ['apple', 'mango', 'cherry', 'orange']
